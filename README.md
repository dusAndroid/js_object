### 面向对象
* 函数：工具-封装了实现细节，不需要了解内部，只要会用就可以了：封装性
* 对象：工具包-把相类似的功能放在一起管理
#### 思路
* 传统开发四要素：
  - 定义变量 保存数据
  - 获取元素
  - 绑定元素
  - 绑定事件
* 对象方式开发:
  - 需要几个工具包
  - 开发工具
  - 使用工具组织代码结构
  - 先宏观思考，再组织代码细节

#### 面向对象的字面量形式-----Json
* 字面量其实就是原型对象的一个实例
* 我们再使用对象的字面量形式时不需要再去实例化

#### JSON协议和JSON对象

1. JSON协议指的是：后台与前端约定后的通信方式
2. json对象与json字符串的相互转换{重点}
- 后台传给前台的是json字符串，前端拿到字符串后需要将字符串转换为json对象即对象的字面量形式
